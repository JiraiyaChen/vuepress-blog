# 2026-02-27 中央主要领导述职摘要

### 会议背景
中央政治局委员、书记处书记，全国人大常委会、国务院、全国政协党组成员及最高法与最高检党组书记按党中央规定，就2025年工作向党中央和习近平总书记书面述职。习近平审阅了报告并提出重要要求。

## 核心内容

### 1. 加强政治责任感和历史使命感
- 坚持以习近平新时代中国特色社会主义思想为指导，深入贯彻党的二十大和二十届历次全会精神。
- 增强“四个意识”、坚定“四个自信”、做到“两个维护”。
- 始终思想、政治与党中央保持高度一致，以实际行动落实中央决策。

### 2. 推动“十五五”首年工作
- 坚持稳中求进总基调，全面贯彻新发展理念。
- 推进高质量发展，统筹发展与安全。
- 持续保障和改善民生，确保社会和谐稳定，实现良好开局。

### 3. 严守中央八项规定
- 积极落实精神与相关细则要求，保持清醒头脑，树立道德自律模范。
- 笃信务实，勇于担当，身体力行践行正确政绩观。
- 努力为人民拿出实绩，用实干创造成果。

### 4. 全面从严治党责任
- 落实从严治党政治责任，坚持不敢腐、不能腐、不想腐。
- 加强制度落实与环境营造，推动形成风清气正的政治生态。

文件内容强调当前形势下的稳进步伐、民生保障与高质量发展的重要性，是未来各级党政机关执行的方向标。
