# 2026年十四届全国人大四次会议江苏代表团发言汇总

## 习近平总书记参加江苏代表团审议
- 时间：3月6日
- 要点：习近平参加江苏代表团审议，强调江苏等经济大省要在研究新情况、解决新问题上下功夫、出经验

## 代表发言要点

### 1. 主题发言（6位代表）
史志军、张晓宏、石磊、杨恒俊、高德荣、张雨霏等6位代表分别围绕以下主题发言：
- 推进新型工业化
- 促进科技创新与产业创新融合
- 加强关键核心技术攻关
- 建设和美乡村
- 破解种业难题
- 发扬体育精神

### 2. 郑海涛（扬州市市长、WCCO主席）
- 主题：做强实体经济，做优名城品质，扎实推进高质量发展

### 3. 吴新明（南通市委书记）
- 主题：学习习近平总书记重要讲话精神，高屋建瓴、论述精辟，为中国式现代化江苏新实践定向领航

### 4. 信长星（省委书记）
- 主题：完全赞同政府工作报告，就抓好"十五五"经济社会发展提出要求

### 5. 其他代表发言
刘小涛、张忠、储永宏、马欣、胡广杰、樊金龙、陈忠伟、王剑锋、张国梁、邢青松等代表也先后发言
