# 2026-03-02 中共中央政治局会议摘要

### 会议背景
中共中央政治局于2月27日召开会议，讨论国务院提交的《中华人民共和国国民经济和社会发展第十五个五年规划纲要草案》和《政府工作报告》稿。习近平总书记主持会议。

## 核心内容

### 1. 十四五期间成就总结
- 面对世纪疫情和复杂环境，经济实力、科技实力显著提升。
- 中国式现代化取得新成就，全面建成社会主义现代化迈出坚实步伐。

### 2. 十五五规划重点
- “十五五”时期是推动社会主义现代化的关键。
- 目标包括巩固优势、补短板、促高质量发展。
- 统筹“五位一体”布局和“四个全面”战略布局，全面贯彻新发展理念。

### 3. 当前政府工作方向
- 强调稳中求进，坚持经济建设为中心，推动高质量发展。
- 加强宏观经济政策前瞻性，持续扩大内需，推进科技自立。
- 关注重点领域风险，实施稳定就业、企业及市场的举措。

### 4. 促进改革开放与民生改善
- 推动重点领域改革及全国统一大市场建设。
- 深化乡村振兴，推进绿色发展与城镇协调。
- 大力保障民生，加强社会稳定与安全建设。

### 5. 政府建设与领导—新阶段
- 提倡改革创新驱动，加强政府自身能力建设。
- 树立正确政绩观，全面从严治党，落实中长期目标。

文件内容反映了新阶段的顶层规划及实践方略，是未来重点聚焦领域的指导蓝图。
