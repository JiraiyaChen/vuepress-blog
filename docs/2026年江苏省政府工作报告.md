# 2026年江苏省政府工作报告

*——2026年2月3日在江苏省第十四届人民代表大会第五次会议上*
*省长 刘小涛*

## 一、"十四五"时期和2025年工作回顾

### "十四五"时期主要成就

| 指标 | 数据 |
|------|------|
| 地区生产总值 | 14.2万亿元 |
| 年均增长 | 5.7% |
| 制造业增加值占比 | 33.5% |
| 全社会研发投入强度 | 3.38% |
| 进出口规模 | 全国第二 |
| 人均可支配收入年均增长 | 6% |

"十四五"时期是江苏发展历程中极不平凡的五年。

## 二、"十五五"时期主要目标

- 地区生产总值年均增长**5%左右**
- 因地制宜发展新质生产力
- 建设现代化产业体系
- 推动绿色低碳转型
- 十项重点任务

## 三、2026年重点工作

1. 巩固经济向好势头
2. 深化改革开放
3. 促进城乡融合发展
4. 统筹发展和安全
5. **支持人工智能创新创业**
6. 城镇新增就业**120万人**
