# 2026年全国两会江苏代表团科技创新和产业创新讲话汇总

## 习近平总书记金句
- "让创新链和产业链无缝对接"
- "强化教育对科技和人才的支撑作用"
- "加快发展新质战斗力"

## 信长星（省委书记）讲话要点

从2024年到2026年，习近平总书记对江苏发展新质生产力的要求一以贯之、与时俱进：

| 年份 | 习近平总书记要求 |
|------|------------------|
| 2024年 | 强调"因地制宜加快发展新质生产力" |
| 2025年 | 强调"在推动科技创新和产业创新融合上打头阵" |
| 2026年 | 强调发展新质生产力要"努力走在前列" |

## 江苏做法

1. **着力打造**具有全球影响力的**产业科技创新中心**
2. **坚持**"既多出科技成果，又把科技成果转化为实实在在的生产力"
3. **推动**传统产业改造升级和开辟战略性新兴产业、未来产业新赛道并重

## 背景

习近平总书记参加江苏代表团审议时，希望江苏在推动科技创新和产业创新融合上"打头阵"，要求江苏等经济大省要在研究新情况、解决新问题上下功夫、出经验。
